# codejuan.com

This repository holds the **published build output** for [codejuan.com](https://codejuan.com),
Juan Contreras' portfolio. It is generated and deployed automatically — do not edit it by hand.

Source lives at **[Code-Juan/codejuan-portfolio](https://github.com/Code-Juan/codejuan-portfolio)**
(an Astro project). To publish, run `scripts/deploy.sh` from the source workspace.
